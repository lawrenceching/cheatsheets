###

```bash
cat hello.txt | grep world
echo "Hello, world!" | grep world
grep world hello.txt
```

### 基础参数

| 参数              | 功能          |
| ----------------- | -------------------  |
| -i   | 不区分大小写              |
| -v   | 输出不符合表达式的文本              |
| -c   | 输出打印了多少行文本              |
| -l   | 输出打印了多少行文本              |
| -n   | 打印行号              |
